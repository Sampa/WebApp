Functional description

Description

Iterates over the entire storage array (9x6) to count occupied spots and segments them by color (Green, Blue, Black). Updates the global inventory statistics in ioRack, ensuring they accurately reflect the matrix's state without nested loops in the main logic.

  • ioRack: In/Out reference to the warehouse rack data structure (typeWarehouseRack).
