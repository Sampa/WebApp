Functional description

(*en-US

Description

Processes all system outputs by executing analog scaling for the crane's X and Z axes using fbIO_AnalogOutput. It evaluates logic for entry, load, and unload conveyors via ConveyorDrv instances, and maps system states to physical actuators and SCADA signals.

Interface Specifications

  • Input: Main system input structure.
  • AvailableSpots: Integer count of free spots.
  • scadaStatusTransfer: Output structure to SCADA.
  • ioOutput: In/Out reference to the main system outputs.
  • ioCrane: In/Out reference to the crane data structure.
  • ioConveyors: In/Out reference to conveyor data.
  • ioState: In/Out reference to machine states.

Safety Interlocks

All physical outputs are securely overridden by the ioState.ActuatorsAllowed permissive. *)
