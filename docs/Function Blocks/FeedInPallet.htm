Functional description

(*en-US

Description

Manages the logic to activate the entry and load conveyors to bring a new pallet into the staging area, preparing it for the LoadSequence.

Interface Specifications

  • Execute: Boolean input; sequence sets this to true when it wants a new pallet.
  • ioInput: In/Out reference to the main input structure.
  • ioOutput: In/Out reference to the main output structure.
  • ioState: In/Out reference to the machine state structure.

Safety Interlocks

  • Operations are interlocked with the ioState.MachineState. *)
