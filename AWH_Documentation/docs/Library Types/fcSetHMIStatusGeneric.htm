fcSetHMIStatusGeneric (FC / 3.0.1)

Overview

Short description

Wrapper function for setting the HMI status of a device.

It will display in order of priority, based on input, where Status1 > Status 2 > Status 3 > Status 4 > Status 5 > Status 6.
