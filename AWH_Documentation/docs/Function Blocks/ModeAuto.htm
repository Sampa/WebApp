Functional description

(*en-US

Description

Coordinates the high-level decision making for automatic operation. It evaluates the current mode selection (Load vs Unload), manages the execution of the FeedInPallet, LoadSequence, and UnloadSequence blocks, and ensures that only one sequence is active at any given time.

Interface Specifications

  • Execute: Boolean input to enable automatic execution.
  • ioInput: In/Out structure for physical inputs.
  • ioOutput: In/Out structure for physical outputs.
  • ioState: In/Out structure tracking machine states.
  • ioCrane: In/Out structure for crane data.
  • ioRack: In/Out structure for inventory matrix.

Safety Interlocks

  • Sequences are mutually exclusive.
  • Sub-sequences handle their own movement safety interlocks. *)
