Functional description

(*en-US

Description

This block takes sensor values, group bools together, run AnalogScaling where applicable and stores the data.

Interface Specifications

  • ReqUnloadMode: Should we ask the system to switch from Loading to unloading the rack (Only used for MachineStateAuto)

  • ioScadaCommands: Input structure for signals recieved from SCADA events.

  • ioSensors: In/Out reference to the main system inputs.

  • ioCrane: In/Out reference to the crane data structure. -fbIO_AnalogInput_Instance: Instance DB for analog input scaling

*)
