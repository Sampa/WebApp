Functional description

Description

Iterates through the 9x6 ioRack.AllStorageSpots matrix to find a matching location based on desired status (empty/occupied) and item type (color code). If a match is found, the physical X and Z coordinates and the array indices are returned for crane positioning and logic usage.

Interface Specifications

  • iSearchForStatus: Integer input specifying the target status (0 = Empty, 1 = Occupied).
  • iSearchForType: Integer input for item color/type (0 = Any, 1 = Blue, 4 = Green, 7 = Black).
  • ioRack: In/Out structure containing the AllStorageSpots 2D array.
  • q_xFoundSpot: Boolean output indicating if a valid spot was located.
  • q_iFoundColor: Integer output showing the actual color type found at that location.
  • q_iTargetPositionXEng: Integer output of the X coordinate in engineering units.
  • q_iTargetPositionZEng: Integer output of the Z coordinate in engineering units.
  • q_iIndexX: Integer output of the array X-index.
  • q_iIndexZ: Integer output of the array Z-index.

Safety Interlocks

None. Returns q_xFoundSpot = FALSE if no match exists, which must be handled by caller.
