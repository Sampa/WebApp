Functional description

Description

Updates the target X and Z position coordinates (in mm) for the crane by writing to the crane's target position data structure.

Interface Specifications

  • X: Integer input specifying the desired X-axis coordinate in mm.
  • Z: Integer input specifying the desired Z-axis coordinate in mm.
  • ioCrane: In/Out reference to the crane data structure (typeCrane) which contains the target positioning properties.
